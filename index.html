Semistrukturierte Daten in relationalen Datenbanksystemen

  • Bachelorarbeit - Thema

Durch die gegenwärtig rasant wachsende Zahl an Mobil- und Web-Applikationen spielen semistrukturierte Daten eine immer größere Rolle. Der klassische Einsatz ist dabei der "einfache Datenaustausch". Relationale Datenbanksysteme sind im Gegensatz auf eine hoch strukturierte Form der Datenspeicherung ausgerichtet. Um die Vorteile dieser Datenbanksysteme und den dort etablierten Möglichkeiten (zB hochperformante Anfragen, Data Warehousing) auch für semistrukturierte Daten nutzen zu können, bieten bekannte Datenbanken wie Oracle DB und MS SQL Server die Möglichkeit semistrukturierte Daten in relationalen Schemen abzubilden.

Ziel meiner BA ist es die Integrationsmöglichkeiten von JSON formatierten Daten in den relationalen Datenbanksystemen von Microsoft und Oracle zu untersuchen. Eingebettete Funktionsumfänge, das Zusammenspiel mit Daten in Relationen, Performance- und Implementierungsunterschiede sollen aufgezeigt und mit Beispielen hinterlegt werden.
